## 📄 <span id="description">Description</span>


# The project description.

This project involves creating a simple calculator with a selection of the desired operation.
Only two numbers are usable.

# Compilation command (exact options)

gcc -Wall -pedantic -Werror -Wextra -std=gnu89

# Run instructions

To run it, type `./calculator`

# Supported operations

Addition, subtraction, multiplication, and division.

# Numer behavior (integer or decimal)

It is possible to use two decimal numbers. The result will have 6 digits after the decimal point.
